## Instructions
This will run and produce the plots and tables in the [arXiv:2510.01328](https://arxiv.org/abs/2510.01328)


In a python environment or conda with pip dependency, run

```
pip install git+https://github.com/DiaaEddinH/DiffusionModels.git@version2
```

This will install the `diffusion_models` package in the environment which is used by the notebook.
